### Installation / How to Run

You can run EncodeX in two ways:

1. **Run the pre-built app (easiest, no setup required):**  
   - Download the `.exe` file from the [Releases](#) section (coming soon).  
   - Double-click to launch the app on Windows.

2. **Build and run from source:**  
   - Clone the repository:  
     ```bash
     git clone https://github.com/ali-hue-byte/EncodeX.git
     ```  
   - Navigate to the `EncodeX` folder.  
   - Open the `EncodeX.slnx` file in **Visual Studio 2022 or later**.  
   - Press **F5** or click **Start** to build and launch the app.

**Requirements for building from source:**  
- Windows OS (WPF apps only run on Windows)  
- [.NET 6 or later](https://dotnet.microsoft.com/) installed  
- Visual Studio 2022 or later recommended
